# Traffic-Racer

___

### Unity Application

---

### Summary

---

An endless platform game where we control a car and avoid the obstacles that come our way.

### Tech Stack

---

* Unity
* C#
* Asset Store

### Requirements

---

* Unity

### Design Patterns

---

* Singleton
* Observer
* Object Pool
